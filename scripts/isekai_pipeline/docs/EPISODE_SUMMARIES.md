# 혈영 이세계편 - 60화 에피소드 서머리

> **이 문서는 대본 작성 전 반드시 확인하는 에피소드별 가이드입니다.**
> 각 화의 핵심 사건, 캐릭터 상태, 복선을 정리하여 스토리 일관성을 유지합니다.

---

## 사용법

대본 작성 전:
1. 해당 에피소드 서머리 확인
2. 이전 에피소드 엔딩 확인 (연결)
3. 다음 에피소드 오프닝 확인 (복선)
4. 캐릭터 경지/상태 확인

## 1부: 이방인 (1~10화)

### EP001 - 이방인

| 항목 | 내용 |
|------|------|
| **핵심 사건** | 이세계 전이, 내공 소실 자각, 생존 시작 |
| **등장인물** | 무영 (단독) |
| **무영 상태** | 내공 없음, 검술 기억만 남음, 언어 불통 |
| **감정선** | 혼란 → 절망 → 냉소적 수용 → 결심 |
| **복선** | 마나의 존재 (아직 인지 못함) |
| **훅/클리프행어** | 낯선 세계에서 홀로 걷기 시작 |

**오프닝 (30초 몽타주):**
- 노비 시절 → 노인에게 무공 전수 → 설하와의 만남 → 혈영검법 완성 → 천마교 최종전 → 차원 균열 → "설하...!"

**본편 구조:**
1. 숲에서 눈뜸 - 낯선 하늘, 낯선 냄새
2. 내공 소실 자각 - 단전이 비어있음, 냉소적 반응
3. 언어 불통 - 지나가는 사람들의 말을 알아듣지 못함
4. 생존 행동 - 물 찾기, 방향 정하기
5. 결심 - 어딘가로 향하며 걷기 시작

**주의사항:**
- 혈마 등장 금지 (33화에 등장)
- 설하는 오프닝 몽타주에서만
- 무영의 독백은 짧고 냉소적으로

### EP002 - 언어의 벽

| 항목 | 내용 |
|------|------|
| **핵심 사건** | 언어 장벽 체감, 카이든과의 첫 만남 |
| **등장인물** | 무영, **카이든 (첫 등장)**, 마을 주민들 |
| **무영 상태** | 내공 없음, 체력 저하, 굶주림 |
| **감정선** | 고립감 → 경계 → 미묘한 호기심 |
| **복선** | 카이든의 선의, 이 세계의 "기사" 개념 |
| **훅/클리프행어** | 카이든이 음식을 건네며 미소 |

**이전 화 연결:**
- 무영이 걷다가 마을에 도착

**본편 구조:**
1. 마을 도착 - 낯선 건축 양식, 사람들의 시선
2. 소통 시도 실패 - 손짓, 표정으로만 의사소통
3. 굶주림 - 돈도 없고, 말도 안 통함
4. 카이든 등장 - 수비대 병사, 동양인 무영에게 호기심
5. 첫 접촉 - 카이든이 빵을 건넴, 무영은 경계하지만 받음

**카이든 첫인상:**
- 밝고 친근한 성격
- "뭐야, 넌 어디서 왔어?" (무영은 못 알아들음)
- 동양인을 처음 봐서 신기해함

**무영의 관찰:**
- 카이든의 검술을 관찰 (수준 파악)
- "...약하군." (내심)

### EP004 - 자유도시

| 항목 | 내용 |
|------|------|
| **핵심 사건** | 자유도시 도착, 모험가 길드 등록 |
| **등장인물** | 무영, 카이든, 길드 직원, 모험가들 |
| **무영 상태** | 내공 없음, 언어 약간 습득 |
| **감정선** | 관찰 → 적응 시도 |
| **복선** | 경지 체계 (일반 기사 → 소드마스터), 마나 개념 |
| **훅/클리프행어** | 등록증에 "E급" 표시 |

**이전 화 연결:**
- 마수를 피해 자유도시에 도착

**본편 구조:**
1. 자유도시 첫인상 - 활기찬 거리, 다양한 종족
2. 길드 도착 - 카이든의 안내
3. 등록 절차 - 간단한 테스트 (체력, 검술 기본)
4. 결과 - E급 판정 (최하위)
5. 무영의 반응 - 냉소적 수용 "...그래, E급이라."

**세계관 설명 (자연스럽게):**
- 모험가 등급: E → D → C → B → A → S
- 기사 경지: 일반 기사 → 엑스퍼트 → 그래듀에이트 → 소드마스터
- 카이든은 일반 기사 수준

4. 설명 - 카이든이 "마나"를 설명
5. 깨달음 - 기와 같은 것, 하지만 농도가 다름

**핵심 대사 (무영 내면):**
- "기다. 이건 기야."
- "농도가... 무림의 열 배는 되겠군."
- "심법으로 이걸 다룰 수 있다면..."

| **감정선** | 기대 → 좌절 → 재시도 결심 |
| **복선** | 심법의 원리, 마나 순환 개념 |

**이전 화 연결:**

**본편 구조:**
1. 혼자 수련 - 숙소에서 심법 운용 시도
2. 첫 시도 - 마나를 끌어오려 함, 실패

5. 미세한 반응 - 단전에서 뭔가 느껴짐

- 여긴 기가 넘쳐남 → 모으는 게 아니라 조절해야
- 심법을 역으로 응용해야 할 듯
