## 추가 개발 계획

### 디자인 카테고리
설교 내용을 기반으로 시각적 자료 자동 생성

| 디자인 유형 | 설명 | 활용처 |
|------------|------|--------|
| PPT 슬라이드 | 설교 포인트별 슬라이드 자동 생성 | 예배 화면 |
| 말씀 카드 | 핵심 성경 구절 + 디자인 이미지 | SNS 공유 |
| 주보 템플릿 | 설교 요약 + 레이아웃 | 주보 제작 |
| 배너 이미지 | 설교 제목 + 배경 이미지 | 유튜브 썸네일, 홍보 |
| 카드뉴스 | 설교 내용 요약 슬라이드 | 인스타, 블로그 |

### 구현 방법 (예상)
1. **텍스트 → 디자인 요소 추출**: GPT로 핵심 문구, 키워드 추출
2. **템플릿 기반 생성**: 미리 만든 템플릿에 텍스트 삽입
3. **이미지 생성 API**: DALL-E, Stable Diffusion 등 활용 (선택)
