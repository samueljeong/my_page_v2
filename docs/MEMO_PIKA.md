# Pika Labs 메모

> 작성일: 2026-01-12
> 상태: 검토 중 (나중에 재검토)

## 용도
- 영상 첫 장면만 AI 영상으로 생성 (5~8초)
- 나머지는 기존 이미지 슬라이드 방식 유지

## 요금제

| 플랜 | 월 가격 | 크레딧 | 영상 개수 (18크레딧 기준) |
|------|--------|--------|------------------------|
| Basic | 무료 | 80 | ~4개 |
| Standard | $8 | 700 | ~38개 |
| Pro | $28 | 2,300 | ~127개 |
| Fancy | $76 | 6,000 | ~333개 |

## 크레딧 사용량
- 이미지→영상 (3초): 6~18 크레딧
- 연장: 1 크레딧/초
- 5초 영상 = 약 20 크레딧

## 비용 비교 (20분 영상 전체)
| 방식 | 비용 |
|------|------|
| Pika | ~$100 |
| Google Veo Fast | ~$180 |
| FFmpeg (현재) | $0 |

## 추천
- **Standard ($8/월)** 이면 월 35개 에피소드 첫 장면 커버 가능
- 전체를 AI 영상으로 만드는 건 비효율적
