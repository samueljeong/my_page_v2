# 아이디어 기록

사용자의 아이디어를 `docs/IDEAS.md` 파일에 기록합니다.

## 입력
$ARGUMENTS

## 실행 방법

1. `docs/IDEAS.md` 파일을 읽습니다
2. 오늘 날짜 섹션이 없으면 새로 만듭니다
3. 아이디어를 체크박스 형식으로 추가합니다:
   ```
   - [ ] [아이디어 내용] (등록: YYYY-MM-DD)
   ```
4. 파일을 저장합니다
5. 사용자에게 "아이디어 기록 완료" 메시지를 보여줍니다

## 형식

아이디어는 다음 형식으로 기록:
- [ ] 아이디어 내용 (등록: 날짜) [상태: 아이디어]

상태 종류:
- 아이디어: 초기 등록
- 검토중: 구현 가능성 검토
- 계획됨: 구현 예정
- 진행중: 구현 중
- 완료: 구현 완료
- 보류: 나중에 재검토

## 주의사항
- 아이디어만 기록하고, 구현은 하지 않습니다
- 기록 후 현재 작업으로 돌아갑니다
